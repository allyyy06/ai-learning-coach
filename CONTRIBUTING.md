# Katkıda Bulunma Rehberi (Contributing Guidelines)

**AI Öğrenme Koçu** projesine katkıda bulunmak istediğiniz için teşekkürler! 🎉

Bu proje açık kaynaklıdır ve her türlü katkıya (hata düzeltmeleri, yeni özellikler, dokümantasyon iyileştirmeleri) açıktır.

## Nasıl Katkıda Bulunabilirsiniz?

1. **Projeyi Fork'layın:** Sağ üst köşedeki "Fork" butonuna tıklayarak projeyi kendi GitHub hesabınıza kopyalayın.
2. **Klonlayın:** Kendi kopyanızı bilgisayarınıza indirin.
   ```bash
   git clone https://github.com/<KULLANICI_ADINIZ>/ai-learning-coach.git
   ```
3. **Yeni Bir Dal (Branch) Oluşturun:**
   ```bash
   git checkout -b ozellik/yeni-harika-ozellik
   ```
4. **Değişikliklerinizi Yapın ve Commit'leyin:**
   ```bash
   git add .
   git commit -m "feat: yeni harika özellik eklendi"
   ```
5. **Dalınızı Push'layın:**
   ```bash
   git push origin ozellik/yeni-harika-ozellik
   ```
6. **Pull Request (PR) Açın:** GitHub üzerinden orijinal depoya bir Pull Request gönderin.

## Kod Standartları
- Proje Python ile yazılmıştır ve `PEP 8` kod standartlarına uymaya gayret ediyoruz.
- Pull Request gönderirken yaptığınız değişiklikleri kısaca açıklayan anlaşılır bir başlık ve açıklama (description) yazmayı unutmayın.

Katkılarınız için şimdiden teşekkürler! 🚀
